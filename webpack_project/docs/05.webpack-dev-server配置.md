# webpack-dev-server配置

热更新，提升开发效率

1. webpack配置webpack-dev-server

  webpack.config.js
   ```
     // 与entry配置同级
     devServer: {
        contentBase: './dist',
        // 自动打开浏览器
        open: true
     }
   ```
   
   package.json
   ```
   "start": "webpack-dev-server",
   ```
