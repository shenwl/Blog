# 使用babel处理ES6语法

## 两个核心库
1. babel-loader
  - 配置rules
  ```
   {
      test: /\.js$/,
      // 排除node_modules下的文件
      exclude: /node_modules/,
      loader: "babel-loader"
    },
  ```
  
  - babel-loader并不会帮你把ES6语法翻译成ES5语法，还需要借助其他模块：@babel/preset-env
  
  - @babel/preset-env 包含了ES6翻译为ES5的规则
 
2. @babel/core
  - 识别js内容转化为抽象语法树（AST），然后编译转化为新的语法
