# 打包React代码

1. [babel preset-react文档](https://www.babeljs.cn/docs/babel-preset-react)


2. .babelrc配置
```json
{
   "presets": [
     ["@babel/preset-env", {
       "targets": {
         "chrome": "67",
         "ie": "9"
       },
       "useBuiltIns": "usage"
     }],
     "@babel/preset-react"
   ]
}

```
- 顺序是从下往上(从右往左)
  - "@babel/preset-react" -> "@babel/preset-env"
