# webpack 核心概念：loader

## 一、什么是loader

一种打包方案，告诉webpack某种文件格式该如何打包。

1. 一个简单的file-loader配置
  ```js
    module.exports = {
      entry: {
        main: './src/index.js',
      },
      module: {
        rules: [
          {
            test: /\.(png|jpg|gif|jpeg)$/,
            use: {
              loader: 'file-loader',
            }
          }
        ]
      },
      output: {
        filename: 'main.js',
        path: resolve(__dirname, './dist'),
      },
    };  
  ```
